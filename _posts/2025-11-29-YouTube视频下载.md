## 下载视频

打开所在文件夹,在此文件夹的地址栏输入**cmd**；

![alt text](https://telegraph-image-9ya.pages.dev/file/AgACAgUAAyEGAATAfiiVAAMKaSweWr-mtgbjiFHD1trBOTr-GGAAAlsLaxvOuGBVPXV-UNNY31kBAAMCAAN5AAM2BA.png)


弹出命令行，输入下方命令；

```
# CMD 命令：yt-dlp.exe+空格+视频网址
yt-dlp.exe https://www.youtube.com/watch?v=xxxxxxxx
```

### 自定义视频下载信息

可以通过下方命令知道视频支持下载哪些清晰度的视频或音质的音频：

```
yt-dlp -F "https://www.youtube.com/watch?v=iZ3b305mx00"
```
正常情况下会出现关于视频的详细信息：

```
ID: 格式的唯一编号，这是你选择格式时需要用到的关键信息。
EXT: 文件扩展名（如 mp4, webm, m4a）。
RESOLUTION: 视频分辨率（如 1920x1080, 1280x720）或音频信息。
FPS: 视频的帧率。
Note: 额外信息，比如 "4K"、"HDR" 或者 "storyboard"。
vcodec/acodec: 视频/音频编码格式。
```

一个完整的视频包括视频和音频，下载时可以选择ID为399的1080p视频 + ID为251的音频；

![alt text](https://telegraph-image-9ya.pages.dev/file/AgACAgUAAyEGAATAfiiVAAMMaSwekyq1ojAU--8qqXOEnQS3_yIAAl0LaxvOuGBVhJJRQvjRuT0BAAMCAAN5AAM2BA.png)



下载此视频自定义画质的命令：

```
yt-dlp -f 399+251 "https://www.youtube.com/watch?v=iZ3b305mx00"
```

## 异常情况：视频/音频合并

正常来说，下载好的视频是一个文件。但如果出现视频和音频分开的情况，就需要使用[ffmpeg工具](https://ffmpeg.org/)来合并媒体文件；


![alt text](https://telegraph-image-9ya.pages.dev/file/AgACAgUAAyEGAATAfiiVAAMNaSwe6Hc_ilmQvb_7Iy6ablSLtYYAAl4LaxvOuGBV8vI-Wgd641UBAAMCAAN5AAM2BA.png)

工具下载完成后，把它放到与yt-dlp相同的文件夹中。

![alt text](https://telegraph-image-9ya.pages.dev/file/AgACAgUAAyEGAATAfiiVAAMPaSwfGMyAfxAvcVIJHRYAAZW8nryIAAJgC2sbzrhgVTeuUBPbH7G_AQADAgADeQADNgQ.png)

在此文件夹的地址栏输入**cmd**；

![](https://telegraph-image-9ya.pages.dev/file/AgACAgUAAyEGAATAfiiVAAMQaSwfJjZ7rZAQGj98nstOsunoZFoAAmELaxvOuGBVKPiZHcnfL5YBAAMCAAN5AAM2BA.png)

它也是命令行工具，无GUI界面，它的合并命令是：

```
ffmpeg -i video.mp4 -i audio.aac -c:v copy -c:a copy output.mp4

-c:v copy：不重新编码视频，无损、速度快
-c:a copy：不重新编码音频
video.mp4：你的视频
audio.aac：你要合进去的音频
output.mp4：输出文件
```

稍等工具合成视频，完成后文件夹中就会出现完整视频了。
